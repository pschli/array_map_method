The Takeaway

When to use the .map() method

  • If you don't intend to modify your existing array
  • To get a new array from an existing array with the same length
  • To perform an operation on every element

How to use the .map() method

  • As the first parameter, each element gives you the option to derive a new value from it
  • As the second parameter, you can use the index of each element
  • As the third parameter you can access the calling array itself which is very helpful if you are chaining array methods
  • Either use an arrow function or a separately defined named function as a callback function parameter of the .map() method.

Don't use the .map() method for side effects

How it looks

let newArray = originalArray((element, index, callingArray) => {
    do something and return the result
})
