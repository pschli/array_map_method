Let's do some exercises

Exercise 1

Create a new array from an original array. Multiply each element with itself. Output the new array with a console.log().

Exercise 2

Generate a new array from the original by squaring every other element and replacing the remaining ones with 0. Output the new array with a console.log().

Exercise 3

Generate a console.log() output for every element of the original array.

Exercise 4

Use the array "people" to create a new array of strings that combine title and lastName, f.ex. "Mr. Müller". Output the new array with a console.log().

Bonus Challenge

Use the array "people" to create a new array of strings that combine title and lastName as in exercise 4. But create an array that only contains speakers. Output the new array with a console.log().
